# 스마트 쓰레기통 Smart trash can 🗑
이 프로젝트는 동아리 수업을 위해 제가 직접 제작한 것으로 복제, 배포, 수정을 금합니다. 또한 사용 시 저작권 및 출처를 의무적으로 표기하여야 하며, 대회 출품 등의 용도로 사용할 수 없습니다.
